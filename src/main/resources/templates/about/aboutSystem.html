1. Введение

Система предназначена для автоматизации процесса онлайн-бронирования билетов в кинотеатре. Пользователи могут просматривать афишу фильмов, выбирать сеансы, бронировать места и управлять своими заказами. Администраторы имеют возможность управлять залами, фильмами, сеансами и справочниками.

Программа написана на языке Java17 в среде IntelliJ IDEA 2024.3.3 и функционирует под управлением операционной системы Windows 10 и выше. Доступ к данным осуществляется с помощью СУБД PostgreSQL 17.3.

2. Режим пользователя

2.1 Стартовая страница

Главная страница приложения с афишей кинотеатра, где отображается краткая информация о каждом фильме (жанр, год выхода, страна производства) и кнопкой «Подробнее» для перехода к деталям фильма и выбора сеанса.

Главная страница приложения

2.2 Авторизация и регистрация

Для работы с системой пользователю необходимо зарегистрироваться или авторизоваться, если аккаунт уже существует.

На странице регистрации пользователь может ввести логин, пароль, имя, фамилию и email, затем нажать кнопку «Зарегистрироваться» для создания аккаунта. Также пользователь может перейти к форме входа, если аккаунт уже существует.

Страница регистрации

На странице входа пользователь может ввести логин и пароль, затем нажать кнопку «Войти» для авторизации. Также пользователь может перейти к форме регистрации, если аккаунта нет.

Страница входа в личный кабинет

2.3 Личный кабинет

В личном кабинете отображается личная информация пользователя (логин, email, имя и фамилия). Пользователь может редактировать данные, просматривать историю бронирования и выйти из аккаунта.

Страница личного кабинета

На странице редактирования личных данных пользователь может изменить логин, email, имя и фамилию, затем нажать кнопку «Сохранить» для применения изменений или «Закрыть» для отмены.

Страница редактирования личных данных

На странице истории бронирования отображается список бронирований с указанием номера, даты и времени бронирования, даты и времени сеанса, названия фильма и стоимости. Пользователь может нажать кнопку «Подробнее» для перехода к информации о заказе или «Отменить» для отмены непрошедшего сеанса.

Страница истории бронирования

2.4 Процесс бронирования

Пользователь может выбрать фильм, сеанс, места в зале и подтвердить бронирование.

В модальном окне выбора даты пользователь может выбрать дату в календаре и нажать кнопку «Выбрать» для подтверждения.

Модальное окно выбора даты

На странице выбора сеанса отображается информация о фильме (название, жанр, год выхода, страна, режиссёр, описание) и список доступных сеансов с датой, временем, ценой билета и номером зала. Пользователь может нажать кнопку «Назад» для возврата к предыдущей странице или «Купить билет» для перехода к покупке.

Страница выбора сеанса

На странице выбора мест в кинозале пользователь может выбрать места, нажимая соответствующие номера, затем нажать кнопку «Выбрать» для перехода к подтверждению билета или «Назад» для возврата к предыдущей странице.

Страница выбора мест

На странице подтверждения брони отображается информация о выбранном фильме, дате, времени, стоимости билета и местах. Пользователь может нажать кнопку «Забронировать» для завершения бронирования или «Назад» для отмены.

Страница подтверждения брони

2.5 Информация о заказе

На странице просмотра информации о заказе отображаются детали заказа: название фильма, дата, время, выбранные места и общая сумма.

Страница информации о заказе

3. Режим администратора

3.1 Управление залами

На странице просмотра залов отображается список залов. Администратор может добавить и редактировать зал, нажав на кнопку «Добавить зал» и «Редактировать» соответственно.

Страница просмотра залов

На странице настройки схемы зала отображается информация о зале (номер, тип, количество рядов и мест) и схема зала. Администратор может добавить или удалить ряд. При нажатии на кнопку с плюсом открывается модальное окно для настройки ценового коэффициента. Администратор может нажать кнопку «Сохранить» для применения изменений.

Страница настройки схемы зала

На странице редактирования зала администратор может изменить тип зала, количество рядов и мест, затем нажать кнопку «Сохранить» для применения изменений или «Назад» для возврата на предыдущую страницу.

Страница редактирования зала

На странице редактирования ряда администратор может изменить коэффициент стоимости ряда и нажать «Сохранить» для применения изменений.

Страница редактирования ряда

3.2 Управление фильмами

На странице просмотра фильмов отображается список всех фильмов с краткой информацией (жанр, год выхода, страна). Администратор может добавлять, редактировать или удалять фильмы. Также он может нажать кнопки «Подробнее» для просмотра сеанса или «Удалить».

Страница просмотра фильмов

На странице добавления фильма администратор может ввести название, выбрать жанр, год выхода, страну, режиссёра, указать длительность, загрузить постер и добавить описание, затем нажать кнопку «Добавить».

Страница добавления фильма

На странице редактирования фильма администратор может изменить информацию о фильме (название, жанр, год, страна, длительность, постер, описание), добавить несколько режиссеров, нажав на кнопку «Добавить режиссера», затем нажать кнопку «Сохранить» для применения изменений.

Страница редактирования фильма (часть 1)
Страница редактирования фильма (часть 2)

3.3 Управление сеансами

На странице добавления сеансов отображается информация о фильме, а также список сеансов, которые администратор может добавлять, редактировать и удалять.

Страница добавления сеансов

На странице редактирования сеансов отображается название фильма, а также детали сеанса, которые администратор может изменить, нажав на кнопку «Обновить».

Страница редактирования сеансов

3.4 Справочник режиссеров

На странице справочника режиссеров отображается список режиссеров, которые администратор может добавлять, редактировать и удалять.

Справочник режиссеров

На странице редактирования режиссера администратор может изменить ФИО режиссера, нажав на кнопку «Обновить».

Страница редактирования режиссера

3.5 Справочник стран

На странице справочника стран отображается список стран, которые администратор может добавлять, редактировать и удалять.

Справочник стран

На странице редактирования страны администратор может изменить название страны, нажав на кнопку «Обновить».

Страница редактирования страны

3.6 Справочник жанров

На странице справочника жанров отображается список жанров, которые администратор может добавлять, редактировать и удалять.

Справочник жанров

На странице редактирования жанра администратор может изменить жанр, нажав на кнопку «Обновить».

Страница редактирования жанра
