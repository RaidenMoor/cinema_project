О системе

Система предназначена для бронирования и покупки билетов в кинотеатр.

С помощью системы в роли "пользователь" пользователю доступны функции работы в личном кабинете, просмотра афиши, бронирования билетов, просмотра истории бронирования.

С помощью системы в роли "администратор" доступны функции работы со справочниками, функции редактирования афиши, функции конструирования залов.
